# Pedro Pathing Visualizer

The Pedro Pathing Visualizer is a visualizer to visualize paths for Pedro Pathing.

## Running Locally

```bash
git clone https://github.com/BeepBot99/pedro-visualizer.git
cd pedro-visualizer
npm install
npm run dev -- --open
```

## Languages & Frameworks

- Svelte/SvelteKit
- HTML
- TypeScript
- Vite

## Technologies

- TailwindCSS
- Dexie.js
- GSAP
- CodeJar
- Highlight.js
- Svelte CodeJar
